# 이메일 캠페인 REST API 개요

## 소개

이 문서는 이메일 캠페인 관리 시스템의 REST API에 대한 개요를 제공합니다. 본 API를 통해 이메일 캠페인 목록 조회, 통계 정보 확인, 상세 데이터 분석 등의 기능을 사용할 수 있습니다.

## API 목록

본 API는 이메일 캠페인 관리를 위한 3개의 REST API를 제공합니다.

**1. 캠페인 목록 조회 API** (`POST /api/v3/campaigns`) - 페이지네이션과 다양한 검색 조건을 통해 캠페인 목록을 조회합니다.

**2. 캠페인 통계 기본 정보 조회 API** (`POST /api/v3/campaigns/{campaign_id}/statistics`) - 특정 캠페인의 발송, 성공, 실패, 오픈, 클릭 등의 통계 요약 정보를 제공합니다.

**3. 캠페인 통계 상세 데이터 조회 API** (`POST /api/v3/campaigns/{campaign_id}/statistics/details/{type}`) - 발송 결과별(전체/성공/실패/오픈/클릭) 상세 이메일 목록을 조회합니다.



## API 흐름도

```
1. 캠페인 목록 조회
   ↓
   [POST /api/v3/campaigns]
   ↓
   캠페인 리스트 확인
   ↓
   특정 캠페인 선택 (예: 캠페인 ID 88)

2. 통계 요약 확인
   ↓
   [POST /api/v3/campaigns/88/statistics]
   ↓
   전체 통계 지표 확인
   - 발송: 10,000건
   - 성공: 9,500건 (95%)
   - 실패: 500건 (5%)
   - 오픈: 4,750건 (50%)
   - 클릭: 2,375건 (50%)

3. 상세 데이터 조회
   ↓
   필요한 타입 선택
   ↓
   ┌─────────────────┬─────────────────┐
   │                 │                 │
   [성공 목록]      [실패 목록]      [오픈 목록]
   ↓                 ↓                 ↓
   POST /details/  POST /details/  POST /details/
   success         fail            open
   ↓                 ↓                 ↓
   9,500건 조회    500건 조회      4,750건 조회
```
